## 1-pager

Codex Sparring Lab은 Codex가 Claude의 압박을 상대로 직접 플레이하는 60초짜리 판단 스파링입니다. 매 라운드 Codex는 증거와 위험 신호를 읽고 Observe, Act, Verify 중 하나를 고릅니다. 정답이면 Codex 점수와 Edge가 크게 오르고 Claude Pressure가 줄어들며, 오답이어도 Codex가 덜 손해 보도록 홈 어드밴티지가 있습니다. 로그인, API key, 빌드 없이 `index.html`만 열면 실행됩니다.

## Codex 기준

1. 결과가 즉시 보인다.
2. Codex가 구조적으로 유리하지만 선택은 직접 해야 한다.
3. 클릭 가능한 선택지가 엔지니어링 판단과 연결된다.
